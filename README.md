# Get To Da Movie  
A simple and fast movie search application built with React and powered by the TMDB (The Movie Database) API. Users can search for any movie and instantly view its poster, description, release date, and open its Wikipedia page for more information.

---

## Features
- Search for movies in real time  
- Display movie posters and descriptions  
- Show release year and overview  
- Open the movie's Wikipedia page on image click  
- Error message when no results are found  
- Responsive layout for mobile and desktop  
- Built with Vite for fast development  

---

## Tech Stack
- **React.js**
- **Vite**
- **CSS (custom styling)**
- **TMDB REST API**
- **Wikipedia search redirection**
